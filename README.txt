To run the service locally:

1. Install virtual environment with "python -m venv venv"
2. Activate the virtual environment:
  - Windows: "venv\Scripts\activate"
  - Mac/Linux: "source venv/bin/activate"
3. Install required libraries with "pip install -r requirements.txt"
4. Run the code: "python main.py"
5. If everything is alright, you will the link in console. Just click on it to try out the service.
6. To stop the service tap Ctrl+C.
